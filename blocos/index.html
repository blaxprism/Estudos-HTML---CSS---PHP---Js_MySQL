blocos de código

Os blocos de código são utilizadas para criação de blocos selecionados por uma tag que não tem função própria
permitindo assim que sejam atribuidos atributos à esse bloco atráves dessa tag

<DIV> </DIV>

A Div é uma tag de seleção de blocos e não causa nenhuma alteração na página por si só
é um tipo de tag de seleção de bloco genérica, podendo ser utilizada em qualquer parte do código
(Diferentemente das outras tags de seleção de bloco)
